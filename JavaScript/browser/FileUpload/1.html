In this example, I explore what the file control, i.e. an input element of type = file works like. Specifically, I look at the value property and the files property.

And then I also explore what is sent with the HTTP request body when we submit the file.

To run the example, click the Show Properties button and then look at the console. See the picture below.

And then, run this example on a web server (so it will post back to itself, otherwise it won't on the file:/// protocol) and click the Submit button and inspect the Network tab to see the request payload. See the picture below.
